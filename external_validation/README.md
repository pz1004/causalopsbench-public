# External Trace Portability Validation

This directory holds the public-dataset validation track for CausalOpsBench.
It is intentionally separate from the native synthetic replay benchmark because
RCAEval RE2 and LBNL FDD traces validate schema portability and diagnostic
robustness, not CausalOpsBench no-op/oracle replay intervention value.

Raw public datasets should be downloaded into `external_validation/data/raw/`.
That path is ignored by git because public dataset licenses and redistribution
terms differ. Processed episodes, predictions, scores, tables, figures, logs,
and manifests are generated by the `cob_ext` modules.

Canonical conversion commands:

```bash
python -m cob_ext.adapters.rcaeval_re2_adapter \
  --raw_dir external_validation/data/raw/rcaeval \
  --systems SS OB TT \
  --output external_validation/data/processed/episodes/rcaeval_re2.jsonl \
  --manifest external_validation/data/processed/manifests/rcaeval_re2_manifest.yaml \
  --max_evidence_spans 40 \
  --seed 20260519
```

```bash
python -m cob_ext.adapters.lbnl_fdd_adapter \
  --raw_dir external_validation/data/raw/lbnl_fdd \
  --tracks RTU SD_AHU \
  --output external_validation/data/processed/episodes/lbnl_rtu_sdahu.jsonl \
  --manifest external_validation/data/processed/manifests/lbnl_fdd_manifest.yaml \
  --window_minutes 240 \
  --stride_minutes 120 \
  --max_windows_per_csv 5 \
  --seed 20260519
```

Minimum successful run:

```bash
python -m cob_ext.runners.run_external_baselines \
  --episodes external_validation/data/processed/episodes/rcaeval_re2.jsonl \
             external_validation/data/processed/episodes/lbnl_rtu_sdahu.jsonl \
  --output-dir external_validation/outputs/baselines \
  --baselines threshold topology_heuristic

python -m cob_ext.runners.run_external_agents \
  --episodes external_validation/data/processed/episodes/rcaeval_re2.jsonl \
             external_validation/data/processed/episodes/lbnl_rtu_sdahu.jsonl \
  --output-dir external_validation/outputs/agents \
  --models ollama:gemma4:26b ollama:gemma4:e4b \
  --policy react-json \
  --max-steps 4 \
  --temperature 0 \
  --num-ctx 8192 \
  --keep-alive 5m \
  --think false \
  --timeout-s 120
```

The external scorer reports detection, RCA, evidence F1, calibration, parsing
success, runtime efficiency, and an optional External Portability Score. It
refuses unsupported native replay metrics.
